JavaScript被禁用或阻止

检测到按钮和键盘事件无法响应

🚨 问题诊断：
由于JavaScript功能被禁用、被安全软件阻止，或者浏览器设置限制，所有交互按钮都无法正常工作。

📋 解决方案（请依次尝试）：

方案1：直接链接访问
点击下方链接直接跳转到系统：
👦
学生模式
直接进入学生仪表板
👨‍👩‍👧‍👦
家长模式
直接进入家长仪表板
方案2：手动输入地址
在浏览器地址栏输入：
http://localhost:3000/dashboard
方案3：启用JavaScript
• Chrome：设置 → 隐私设置和安全性 → 网站设置 → JavaScript → 允许
• Edge：设置 → Cookie和网站权限 → JavaScript → 允许
• Firefox：地址栏输入 about:config → javascript.enabled → true
方案4：检查安全软件
• 暂时关闭360、电脑管家等安全软件
• 检查浏览器插件（如广告拦截器）是否阻止了JavaScript
• 尝试使用浏览器的无痕/隐私模式

🔧 手动设置用户信息（高级用户）：

如果直接访问dashboard显示未登录，请：

  1. 按F12打开开发者工具
  2. 切换到"控制台(Console)"标签
  3. 粘贴以下代码并按回车：
localStorage.setItem('auth_token', 'demo-token-student-' + Date.now()); localStorage.setItem('user_email', 'student@example.com'); localStorage.setItem('user_role', 'student'); localStorage.setItem('user_display_name', '演示学生'); window.location.href = 'http://localhost:3000/dashboard';
